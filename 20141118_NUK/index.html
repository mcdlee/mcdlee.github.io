社區醫療群

  • 時空背景：
    • 1986年3月成立台灣家庭醫學會
    • 1995年3月，全民健保開辦
    • 大約2003年開始家庭醫師整合照護計畫
  • 5~10個基層診所 + 至少一家醫院
    • 建立家戶健康檔案
    • 建立雙向轉診模式
    • 社區健康營造

執行成效

  • 2011年6月為止，356 個醫療群，參與診所佔全國診所約21.46%，參與醫師佔全國 18.59%。
  • 2008年新參加與未參加民眾相比：門診費用節省5.4%，住院費用節省7.5%。
  • 66.3%民眾認為可節省自行前往醫院的等候時間。
  • 80.6%民眾病情穩定後願意回家庭醫師診所追蹤。
  • 九成民眾表達以後有健康問題會先諮詢家庭醫師。
  • 成人健檢及子宮頸抹片檢查率較一般民眾普及。

臺灣醫界2011 Aug, 54(8):56-60

到處都是「沒有人」

  • 陳嘉葳和陳奎銘兩位大大推薦我用 R 的 ggmap 套件，可以把地址透過 google geocode 轉成座標
  • 他們也推薦我用 leafletR 將座標資料以 leaflet 的方式呈現
  • 有了初步的成果，雖不滿意但可以接受。
  • 二月底的 g0v hackthon 正好有台南場，我就去展示一下順便揪咖。
  • 一整天的活動結束之後，我發現這次收集到的資料其實有些問題。
  • 三月底的 Kaohsiung useR! meetup 聽到 rCharts，裡面有一個功能是產生 leaflet 地圖，而且比 leafletR 適合跟 shiny 整合，也比較有彈性。
  • 清明假期把 rCharts 的 sample code 啃完然後幾乎達成完全體。Source code
  • 兩個多月前，在高雄醫學大學的協助下，這個專案有了個家： http://community.kmu.edu.tw"
