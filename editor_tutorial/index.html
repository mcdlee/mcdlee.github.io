iD 不是我的最愛

iD 的優點

  • 在大多數最新的瀏覽器皆可執行 (包含 IE 和 Safari on iPad)
  • 介面惹人愛
  • 可滿足大多數的需求

iD 相對於 JOSM 的弱點

  • 無法調解編輯衝突
  • Validator 涵蓋不夠完整
  • 沒有過濾器的功能
  • 編輯關係仍不算方便
  • 可以自訂的功能比較少 (eg: 預設組合、快速鍵)

請下載並安裝 JOSM

JOSM 和 iD 操作方式的不同

  • 需額外安裝程式
  • 離線版的編輯器 (offline editor)
  • 用右鍵移動地圖
  • 有很多很多 plug-in
